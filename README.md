## 개요
> 콘서트 좌석 예매 프로그램

- 콘서트 선택 -> 날짜 / 시간 선택 -> 좌석 선택 -> 결제 
- PieChart, BarChart 를 이용한 시각화
- 회원 게시판


## 개발 시간

> ~11/22

## 개발 환경

> Eclipse, Mysql, Scene Builder, Onenote 

## 역할 분담

> 동진 : 홈페이지 가입, 메인 페이지

> 지형 : 매니저 페이지, 메인 페이지

> 정진 : 메인 페이지
